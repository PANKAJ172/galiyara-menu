# üöÄ QUICK START GUIDE - Deploy to GitHub Pages

## ‚ö° Fast Track (5 Minutes)

### Step 1: Create GitHub Account (if you don't have one)
- Go to: https://github.com/signup
- Enter email, password, username
- Verify your account

### Step 2: Create New Repository
1. Click the **"+"** button (top right) ‚Üí **"New repository"**
2. Repository name: `galiyara-menu`
3. Description: "Digital menu for Galiyara Food Court"
4. Make it **Public** ‚úÖ
5. Click **"Create repository"**

### Step 3: Upload Files
1. On the new repository page, you'll see **"uploading an existing file"** link - click it
2. Drag and drop **index.html** file
3. Scroll down, click **"Commit changes"**

### Step 4: Create Menus Folder
1. Click **"Add file"** ‚Üí **"Create new file"**
2. In the filename box, type: `menus/info.txt`
3. In the content box, type: "PDF menu files will be uploaded here"
4. Click **"Commit changes"**

### Step 5: Enable GitHub Pages
1. Click **"Settings"** tab (top of page)
2. Click **"Pages"** in the left sidebar
3. Under "Branch", select **"main"**
4. Click **"Save"**
5. ‚úÖ Done! Wait 2-3 minutes

### Step 6: Get Your Website URL
Your website will be live at:
```
https://YOUR-GITHUB-USERNAME.github.io/galiyara-menu/
```

Replace `YOUR-GITHUB-USERNAME` with your actual GitHub username.

## üÜò Common Issues & Solutions

### ‚ùå Website shows 404 error
**Solution**: 
- Wait 5 more minutes (GitHub Pages can take up to 10 minutes)
- Check that your file is named **exactly** `index.html` (not `Index.html` or `index.HTML`)
- Make sure repository is set to **Public**

### ‚ùå PDF menu not showing
**Solution**:
- Check PDF is in the `menus/` folder
- Verify filename matches exactly (case-sensitive!)
- Make sure PDF is not corrupted

### ‚ùå Changes not appearing
**Solution**:
- Wait 2-3 minutes after committing changes
- Clear browser cache (Ctrl + Shift + R on Chrome)
- Try in incognito/private browsing mode

### ‚ùå QR Code not working
**Solution**:
- Test the URL in your browser first
- Make sure you copied the full URL with `https://`
- Regenerate the QR code if needed
